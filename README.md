# AgroPredict AI — Landing/Dashboard demo

Interfaz frontend con datos simulados. Sin backend, sin APIs, sin ML.

## Stack
- Next.js 14 (App Router)
- React 18 + TypeScript
- Tailwind CSS 3

## Correr local
```bash
npm install
npm run dev
```

## Deploy en Vercel
1. Subí el repo a GitHub
2. En vercel.com → New Project → importá el repo
3. Framework preset: Next.js (auto-detectado) → Deploy

No requiere variables de entorno.
